# 🌊 Aqua_Solar

A floating solar power generation system integrated with water quality monitoring.

## 📌 Description
Aqua_Solar is designed to address land usage issues caused by traditional solar panel installations. Instead of occupying land, this system enables solar panels to float on water bodies while simultaneously monitoring the health of the water.

This dual-purpose approach not only generates renewable energy but also ensures that the water ecosystem remains safe and sustainable.

## 🚀 Features
- ☀️ Floating solar panel setup for power generation  
- 💧 Integrated water quality monitoring system  
- 🌍 Efficient land usage by utilizing water surfaces  
- 📊 Sensor-based data collection (pH, turbidity, etc.)  
- ⚡ Sustainable and eco-friendly solution  

## 🛠️ Tech Stack
- Arduino  
- C++  
- Water quality sensors (pH, turbidity, temperature)  
- Solar panel setup (floating structure)  

## ⚙️ How It Works
1. Solar panels are mounted on a floating structure over water  
2. Panels generate electricity from sunlight  
3. Sensors placed in water collect quality parameters  
4. Arduino processes and outputs the data  
5. System ensures both energy generation and water monitoring  

## ▶️ How to Run
1. Set up the floating structure with solar panels  
2. Connect sensors to Arduino  
3. Upload the code using Arduino IDE  
4. Power and test the system  
5. Monitor output through serial monitor/display
