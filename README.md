# Aqua Solar Water Quality Monitoring System

## Overview
A floating solar-powered platform designed to monitor water quality parameters using an Arduino-based sensing system.

## Parameters Monitored
- Temperature
- Humidity
- Total Dissolved Solids (TDS)

## Technologies
- Arduino
- Embedded C
- Environmental Sensors

## Applications
- Water quality monitoring
- Aquaculture
- Environmental sensing

## Future Improvements
- IoT-based remote monitoring
- Cloud data logging
- AI-based anomaly detection
